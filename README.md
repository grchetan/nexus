## рҹ§  Tech Stack вҖ” And Why I Chose It

### Frontend
```
React 18     вҶ’  Component-based UI вҖ” makes the app modular and maintainable
Vite         вҶ’  Lightning-fast build tool with instant hot reload in development
Context API  вҶ’  Global state management (user session, notifications, page routing)
CSS Variables вҶ’ Consistent design system вҖ” theming without any extra library
```

### Backend
```
Node.js + Express  вҶ’  Lightweight REST API server вҖ” fast and flexible
MongoDB + Mongoose вҶ’  Document-based database вҖ” flexible schema for evolving data
JWT                вҶ’  Token-based auth вҖ” stateless, secure, survives page refreshes
Multer             вҶ’  Handles photo and video file uploads on the server
Bcryptjs           вҶ’  Password hashing вҖ” plain text passwords are never stored
Nodemon            вҶ’  Auto-restarts the server on every file save during development
```

### Why the MERN Stack?

1. **One language everywhere** вҖ” JavaScript on both frontend and backend
2. **MongoDB's flexible schema** keeps the codebase clean as the app evolves
3. **React + Vite** is the fastest developer experience available in 2024
4. **MERN developers are in high demand** across the industry

## вҡҷпёҸ Running It Locally вҖ” Step by Step

### Prerequisites

- [Node.js v18+](https://nodejs.org)
- [MongoDB Community](https://mongodb.com/try/download/community)
- [Git](https://git-scm.com)

---

### Step 1 вҖ” Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/nexus-app.git
cd nexus-app
```

---

### Step 2 вҖ” Create the backend `.env` file

Navigate into the `backend/` folder and create a file named `.env`, then paste this:

```env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/nexus_db
JWT_SECRET=any_long_random_string_you_choose
JWT_EXPIRES_IN=7d
CLIENT_URL=http://localhost:5173
```

> вҡ пёҸ Never push your `.env` file to GitHub вҖ” it contains your secret keys.

---

### Step 3 вҖ” Install all dependencies

```bash
# Root
npm install

# Backend
cd backend && npm install && cd ..

# Frontend
cd frontend && npm install && cd ..
```

---

### Step 4 вҖ” Start MongoDB

```bash
mongod
```

---

### Step 5 вҖ” Start the app (2 terminals)

```bash
# Terminal 1 вҖ” Backend
cd backend
npm run dev
```

```bash
# Terminal 2 вҖ” Frontend
cd frontend
npm run dev
```

---

### вң… You're all set! Open these in your browser:

| | URL |
|---|---|
| рҹҢҗ App | `http://localhost:5173` |
| рҹ”§ API | `http://localhost:5000` |
| вқӨпёҸ Health Check | `http://localhost:5000/api/health` |

> рҹ’Ў **Live reload is already configured.** Save any file in `backend/` and the server restarts automatically. Save any file in `frontend/` and the browser updates instantly вҖ” no manual restarts needed.

## рҹ“Ў API Reference

### рҹ”җ Auth Routes

| Method | Endpoint | What it does |
|---|---|---|
| `POST` | `/api/auth/signup` | Creates a new user account |
| `POST` | `/api/auth/login` | Logs in and returns a JWT token |
| `GET` | `/api/auth/me` | Returns the currently logged-in user's data |

### рҹ‘Ө User Routes

| Method | Endpoint | What it does |
|---|---|---|
| `GET` | `/api/users/suggestions` | Returns "People you may know" list |
| `GET` | `/api/users/search?q=name` | Searches users by name |
| `GET` | `/api/users/:id` | Returns any user's public profile |
| `PUT` | `/api/users/profile` | Updates your own profile |
| `POST` | `/api/users/avatar` | Uploads a profile photo |
| `POST` | `/api/users/cover` | Uploads a cover photo |
| `POST` | `/api/users/:id/connect` | Sends or cancels a connection request |

### рҹ“қ Post Routes

| Method | Endpoint | What it does |
|---|---|---|
| `GET` | `/api/posts/feed` | Returns all posts for the home feed |
| `POST` | `/api/posts` | Creates a new post |
| `PUT` | `/api/posts/:id/like` | Toggles like on a post |
| `POST` | `/api/posts/:id/comment` | Adds a comment to a post |
| `DELETE` | `/api/posts/:id` | Deletes your own post |
